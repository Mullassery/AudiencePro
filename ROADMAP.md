# ClusterAudienceKit Roadmap

## Current Release: 0.1.0

### Completed ✓
- [x] RFM segmentation (Recency, Frequency, Monetary)
- [x] K-Means clustering
- [x] K-Prototypes (mixed numeric/categorical)
- [x] Customer Lifetime Value (CLV) estimation
- [x] Segment profiling and characterization
- [x] Drift detection between time periods
- [x] Streaming segment updates
- [x] Multi-format support (CSV, Parquet, DataFrame)

### Upcoming: 0.2.0
- [ ] Advanced clustering (DBSCAN, Hierarchical)
- [ ] Customer propensity modeling
- [ ] Churn prediction integration
- [ ] A/B testing for segments
- [ ] SQL dialect support (DuckDB, Polars SQL)

### Future Roadmap
- [ ] dbt macros for workflow integration
- [ ] Looker block for visualization
- [ ] Cloud data warehouse connectors (Snowflake, BigQuery, Redshift)
- [ ] Real-time serving API
- [ ] Multi-touch attribution
- [ ] Web dashboard with segment explorer
